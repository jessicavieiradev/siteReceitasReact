# 🍳 Site de Receitas com React

Projeto pessoal desenvolvido para **praticar e demonstrar habilidades em React, TypeScript e TailwindCSS**, além de exercitar o consumo de **APIs**, gerenciamento de estado com **hooks** e navegação com **rotas**.

---

## 🔗 Acesso ao Site

🌐 Acesse o site online pelo GitHub Pages:  
[👉 Ver Site](https://jessicavieiradev.github.io/siteReceitasReact/)  <!-- Substitua # pelo link real do GitHub Pages -->

---

## ✨ Funcionalidades

- 🔍 Buscar receitas por nome  
- 📋 Visualizar ingredientes e modo de preparo  
- 📄 Paginação de receitas
